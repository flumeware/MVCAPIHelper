MVC API Helper
----------------

MVC API Helper helps you to create API's that support:
 - JSON
 - XML
 - CSV
 - JSV
 
for their responses. 
It can also help:
 - Handle Quotas
 - Authentication and Authorization

Nuget Package
------------
MVC API Helper is avalible on Nuget at: http://nuget.org/packages/MVCAPIHelper/
